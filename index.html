What is TypeScript?

  • TypeScript is typed superset of Javascript
  • TypeScript compiles to Javascript
  • Developed and maintained by Microsoft
  • Opensource
  • Documentation www.typescriptlang.org

Static type checking


						let two: number = 2;
						two = '2'; // Error: cannot assign 'string' to 'number'
  • Allows IDEs to perform live error checks
  • Exposes auto-completion and code hinting
  • Easier refactor process
  • Less unit testing

Interface


						interface IName {
							propName1[:type]; // required property
							propName2?[:type]; // optional property
							[propName: string][:type]; // extra properties
						}
  • Design time feature (no extra code would be emitted to JS file
